📊 Excited to share my third project as a Data Science Intern at DecodeLabs!

🔍 Project Title: Exploratory Data Analysis (EDA)

In this project, I performed an in-depth Exploratory Data Analysis on a real-world Vehicle Dataset to uncover hidden patterns, trends, and meaningful insights by:

✔ Calculated basic statistics — mean, median, mode, standard deviation for key numerical columns like price, mileage, and engine size

✔ Analyzed the distribution of vehicle types, fuel types, and transmission categories

✔ Identified trends between vehicle age and price depreciation

✔ Detected outliers in mileage and pricing using statistical methods

✔ Examined correlations between features to understand relationships (e.g., mileage vs. price)

✔ Grouped and compared data by brand, fuel type, and year to find patterns

✔ Generated frequency counts and value distributions for categorical columns

✔ Summarized key findings and observations from the dataset

🔍 Key Outcome:
The EDA revealed meaningful insights about the vehicle market with:
✅ Clear understanding of price distribution across different vehicle categories

✅ Identified top trends in fuel type preferences and vehicle age

✅ Detected significant outliers that could impact future model performance

✅ Strong correlation insights between mileage, year, and vehicle pricing
